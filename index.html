"Formbar.js" 2023-Present

The Formbar is an open source classroom management tool designed by my teacher. This is a tool that allows the teacher to create and control classes, create polls and questions for students to answer and view "Break Requests" and "Help Tickets." This tool was created to manage the classroom easily through a website. For example, a student can send a help ticket to the teacher which will notify them that they require assistance on their work. So far I have worked on fixing bugs and implementing quality of life fixes. It is planned that one day every class in the school will have access and be able to make use of the Formbar website.
